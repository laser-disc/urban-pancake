# urban-pancake

Urban Pancake
  * location data
  * Pull food truck info from Twitter
  * Display locations on map
  * Directions

Tech Stack:
  * Builders - Gulp/Webpack
  * Style Guides - MKS, Karma
  * Front-End - React/Redux
  * Server - Node/Express
  * Tests - Mocha/Chai/Enzyme
  * DB - MySql/neo4J
  * APIs - Twitter/Google Maps/Yelp
  * Deployment - Heroku
